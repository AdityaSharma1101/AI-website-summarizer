# 🧠 AI Website Summarizer with LLM (Streamlit + Ollama + Mistral)

A simple AI-powered web app that takes a website URL (like Wikipedia) and summarizes its content into bullet points, paragraphs, and key insights — explained in a human-like tone using a locally running LLM via Ollama.

---

## 🚀 Features

- 📎 Paste any website URL (especially long-form pages like Wikipedia)
- 🔍 Extracts and cleans webpage content
- 🤖 Uses **Mistral 7B** (via Ollama) to summarize the text like a human
- 📑 Outputs summaries in structured format (points, paragraphs, etc.)
- 🖥️ Built with **Streamlit** for a smooth web experience

---

## 🧱 Tech Stack

- 🐍 Python 3.10+
- 🧠 [Ollama](https://ollama.com/) (to run LLM locally)
- 🧪 Mistral 7B model
- 🌐 Streamlit (for the web interface)
- 🥣 BeautifulSoup (for scraping website content)
